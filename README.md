# aiffel7_library

## 사용해보기

- 패키지 설치

```pip install --upgrade aiffel7```

- example 폴더에서 사용법 확인하기

구현된 함수에대한 문서들이 들어있습니다.

## 기여하기

- pull request로 패키지에 포함할 파일을  aiffel7 폴더에 추가
- 코드 상단에 기여자 적기(날짜 생략 가능)
```python
# 기여자: [기여자 이름]
# 기여 날짜: [YYYY-MM-DD]
# 설명: [변경 사항에 대한 간략한 설명]
# 설명에는 사용처,  이것이 주는 이점 , 원리 등등을 자세히 써주면 좋음
```
- example folder에 사용법 작성
